Verwenden Sie den Piwigo Client auf Ihrem Android-Gerät? Wenn ja, wird dieses Plugin mehr von der standardmäßigen Funktionalität der Piwigo-Website der PiwigoClient App zur Verfügung stellen.

Benötige ich dieses Plugin?

? Dieses Plugin stellt Funktionen innerhalb von Piwigo zur Verfügung, die über die Standardwebsite verfügbar sind, aber von den Webservices weggelassen wurden. Clients wie PiwigoClient (Android) verwenden, um das Surfen und Verwalten auf alternative Weise zu ermöglichen. Im Idealfall werden alle Funktionen dieses Plugins in einer zukünftigen Version auf dem Piwigo-Hauptserver hinzugefügt, aber ich habe keine Kontrolle darüber, ob und wann dies geschehen wird. Bis es das tut, umgeht dieses Plugin dieses Problem.

Features, die dieses Plugin hinzufügt:

  • Eine eigene Bewertung für Fotos anzeigen
  • Fotos als persönlichen Favoriten markieren oder nicht markieren
  • Löschen Sie Ihre Favoritenliste
    • . Protokollieren Sie den Zugriff auf Ihren Piwigo-Server durch Benutzer der WS Functions API (z.B. PiwigoClient Benutzer) .
  • Betrachten und verwalten Sie verwaiste Fotos
    • .

Wichtige Anmerkung

> Wenn Sie die Nutzung der Piwigo Client App einstellen, möchten Sie vielleicht auch dieses Plugin deaktivieren oder von Ihrem Server löschen.
