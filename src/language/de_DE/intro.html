Verwenden Sie Piwigo Client auf Ihrem Android-Gerät? In diesem Fall stellt dieses Plugin der PiwigoClient-App mehr Standardfunktionen der Piwigo-Website zur Verfügung.

Benötige ich dieses Plugin? Dieses Plugin stellt Funktionen in Piwigo zur Verfügung, die über die Standardwebsite verfügbar sind, jedoch in den Webdiensten weggelassen wurden Clients wie PiwigoClient (Android) ermöglichen das alternative Surfen und Verwalten. Im Idealfall werden alle in diesem Plugin enthaltenen Funktionen in einer zukünftigen Version dem Piwigo-Hauptserver hinzugefügt, aber ich habe keine Kontrolle darüber, ob oder wann dies geschehen wird. Bis dahin umgeht dieses Plugin dieses Problem.

Mit diesem Plugin werden folgende Funktionen hinzugefügt:
  • Zeigen Sie Ihre eigene Bewertung für Fotos an
  • Markieren oder deaktivieren Sie Fotos als persönlichen Favoriten
  • Löschen Sie Ihre Favoritenliste
  • Zeichnen Sie den Zugriff von Benutzern der WS-Funktions-API (z. B. PiwigoClient-Benutzer) auf Ihren Piwigo-Server auf.
  • Verwaiste Fotos anzeigen und verwalten

    Wichtiger Hinweis Wenn Sie die Piwigo Client-App nicht mehr verwenden, möchten Sie möglicherweise auch dieses Plugin von Ihrem Server deaktivieren oder löschen.
