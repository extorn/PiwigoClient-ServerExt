Est-ce que vous utilisez Piwigo Client sur votre appareil androïde ?

Est-ce que vous utilisez Piwigo Client ? Si c'est le cas, ce plugin exposera une plus grande partie des fonctionnalités standard du site Web Piwigo à l'application PiwigoClient.

Ai-je besoin de ce plugin ? <

? Ce plugin expose les fonctionnalités de Piwigo qui sont disponibles sur le site web standard, mais qui ont été omises des services web qui tels que PiwigoClient (Android) permettent la navigation et l'administration d'une autre manière. Idéalement, toutes les fonctionnalités incluses dans ce plugin seront ajoutées au serveur principal de Piwigo dans une prochaine version, mais je n'ai aucun contrôle sur si et quand cela arrivera. En attendant, ce plugin contourne ce problème.
