Version du plug-in

1.0.17

Utilisez-vous Piwigo Client sur votre appareil Android?

Si tel est le cas, ce plugin exposera davantage les fonctionnalités du site Web Piwigo standard à l'application PiwigoClient.

Ai-je besoin de ce plugin?

Ce plugin expose les fonctionnalités de Piwigo qui sont disponibles via le site Web standard, mais qui ont été omises des services Web qui des clients tels que PiwigoClient (Android) utilisent pour permettre la navigation et l'administration d'une manière alternative. Idéalement, toutes les fonctionnalités incluses dans ce plugin seront ajoutées au serveur principal Piwigo dans une prochaine version, mais je n'ai aucun contrôle sur si cela se produira ou quand. Jusqu'à ce que ce soit le cas, ce plugin contourne ce problème.

Fonctionnalités ajoutées par ce plugin:

  • Afficher votre propre note pour les photos
  • Marquer ou décocher les photos comme favoris personnels
  • Effacez votre liste de favoris
  • La prise en charge du plug-in de descriptions étendues a fourni des descriptions différentes ou étendues d'albums et de ressources lors de leur consultation
  • Enregistrer l'accès à votre serveur piwigo par les utilisateurs de l'API WS Functions (par exemple les utilisateurs PiwigoClient)
  • Afficher et gérer les photos orphelines
  • Prise en charge de l'identification des sites de synchronisation piwigo non par défaut (c'est-à-dire pas des «galeries») - nécessaire si vous utilisez le plugin de confidentialité piwigo et les fichiers multimédias
  • Prise en charge de la reconnaissance du fait que l'utilisateur récupère une liste de catégories ou d'images et s'attend à être connecté (réponse HTTP 403 si en réalité une liste d'invités publique aurait été renvoyée)
  • Assistance pour la recherche de formats alternatifs pwg_format

Tous les problèmes, demandes de fonctionnalités ou modifications peuvent être soulevés ici projet github

Remarque importante

Si vous arrêtez d'utiliser l'application Piwigo Client, vous pouvez également désactiver ou supprimer ce plugin de votre serveur.
